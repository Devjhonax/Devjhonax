# Olá, eu sou o Jhonathas! 👋

**`Desenvolvedor Full-stack | Foco em Back-end`**

Tenho 18 anos, sou de Camamu – BA e sou apaixonado por construir soluções que resolvem problemas reais. Estou me formando em Desenvolvimento Web Full-stack pelo programa **Programadores do Amanhã**, onde aprendo na prática com projetos autorais, trabalho em equipe e metodologias ágeis.

Meu foco principal é o **Back-end**: arquitetura de APIs RESTful, modelagem de banco de dados, autenticação, segurança e integrações. Também tenho experiência com Front-end utilizando React, HTML, CSS e JavaScript.

Atualmente atuo como **Jovem Aprendiz na Caixa Econômica Federal**, onde desenvolvo habilidades de organização, responsabilidade e comunicação em ambiente corporativo.
